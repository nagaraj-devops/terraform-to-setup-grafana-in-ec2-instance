Steps
1. Get AWS Credentials
2. Create main.tf
3. Port 3000 for grafana
4. commands to download grafana from repo
5. terraform init, validate, plan and apply
6. go for t3.small ubuntu instance with 10Gb EBS
7. AWS Role Creation to allow AWS CLoud Watch
8. Get AWS Secret credentials
